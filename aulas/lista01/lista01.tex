\item
Dada uma máquina PRAM com $n$ processadores, temos na memória compartilhada uma variável C e um vetor A de $n$ posições.  Um determinado algoritmo paralelo precisa, em um dos seus passos, atribuir o valor de C para cada um dos $n$ elementos de A. Suponha que $n$ é uma potência de 2 ou que $n = 2^k - 1$.

Em um modelo CREW, isto é feito da forma abaixo, com leitura simultânea na variável C:

\begin{codebox}
  \zi \For $1 \leq i \leq n$ \DoPar 
  \zi     $A[i] := C$
  \zi  \End   
\end{codebox}

\begin{enumerate}
	\item [a)] Apresente as complexidades do algoritmo acima e diga se ele é ótimo.
	\item [b)] Escreva um algoritmo paralelo que realize a mesma tarefa que o algoritmo acima, porém para o modelo EREW. Apresente as complexidades do algoritmo.
	\item [c)] Reescreva o algoritmo do item (b), de maneira que ele seja ótimo.
	
\end{enumerate}

	\item [b)] Para o modelo EREW, de maneira que o algoritmo seja ótimo.	
\end{enumerate}
